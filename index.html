Ich bin Dustin — Fachinformatiker für Anwendungsentwicklung aus Hamburg. Ich entwickle Software als System aus Logik, Struktur und Interaktion. Mein technischer Hintergrund reicht von Softwareentwicklung, Mechatronik und CAD bis hin zu Robotik und Systemintegration, wodurch ich Probleme ganzheitlich analysiere.

ROBOTIC PROJECT
PROJECT_01
RC-Rover
Eigenentwickeltes mobiles Robotiksystem auf Basis eines Raspberry Pi 4 mit schwenkbarer Kameraeinheit zur visuellen Zielerfassung. Das System kombiniert mehrere Sensormodalitäten zur Umgebungswahrnehmung und Sicherheitsüberwachung: Ultraschallsensorik zur Distanzbestimmung des aktuell visuell anvisierten Objekts, Infrarot-Distanzsensoren zur aktiven Kollisionsvermeidung, Gassensoren zur Detektion potenziell gefährlicher Luftzusammensetzungen sowie Flammensensorik zur Erkennung offener Feuerquellen. Die Architektur ist auf Echtzeit-Datenverarbeitung, Sensordatenfusion und eine stabile Remote-Control- sowie Telemetrieanbindung ausgelegt. Ziel ist die zuverlässige Kopplung von visueller Zielverfolgung und sicherheitskritischer Umgebungsanalyse in einem kompakten robotischen System.
Raspberry pi 4 / Python / SQL / Sensorik
Realtime IoT Robotics

PROJECT_02
Hex & Binär Lernapp
Desktop-Lernanwendung zur Analyse und Umrechnung von Hexadezimal- und Binärwerten. Entwickelt in Python mit Tkinter, inklusive eigener UI-Komponenten und kontrollierter Eingabelogik. Schwerpunkt ist die visuelle Darstellung von Bitstrukturen sowie die schrittweise Nachvollziehbarkeit der Umrechnung zwischen Zahlensystemen in Echtzeit.
Python / TKinter / Desktop App
Bitoperationen Education UI
COMPLETED
PROJECT_03
Netbox Formular System
NetBox-Plugin zum Aufbau und Betrieb strukturierter Formular-Workflows — komplett per Drag & Drop, ohne technisches Vorwissen.NetBox-Plugin zum Aufbau und Betrieb strukturierter Formular-Workflows — komplett per Drag & Drop, ohne technisches Vorwissen. Das System umfasst ein eigenständiges Changelog, das lückenlos aufzeichnet wer wann welches Feld verändert hat und was durch was ersetzt wurde. Formulare können ausgefüllt, versioniert und über eine Git-Diff-ähnliche Vergleichsansicht gegenübergestellt werden — einzeln oder zwischen mehreren Objekten. Renewal Policies steuern intervallbasierte Wiedervorlagen inklusive Warn-Widget. Ein Customer Portal ermöglicht die direkte Weitergabe von Formularen an Kunden zur eigenständigen Befüllung.
Python / Django / HTML / CSS / JavaScript
NetBox Drag & Drop Workflow
LIVE
PROJECT_04
Portfolio System v2
Vollständig in HTML, CSS und JavaScript entwickeltes Portfolio — kein Framework, keine externe Bibliothek. Interaktive Canvas-Animationen, scrollgetriggerte Transitionen und eine modulare Section-Architektur bilden das technische Fundament. Jede visuelle Komponente — vom Hex-Grid-Entry über den Radar-Scanner bis zum Matrix-Terminal — ist von Grund auf selbst implementiert. Das Ergebnis ist diese Seite.
HTML / CSS / JavaScript / ♥ Love
